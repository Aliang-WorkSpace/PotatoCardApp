# 🥔 PotatoCardApp

一个面向 **土豆片-电子拍立得（一款6色墨水屏设备）** 的信息卡片展示 App，支持高质量图片传输与优化显示。

## ✨ 项目介绍

PotatoCardApp 是一个 iOS 应用，主要用于：

- 将图片/信息卡片传输到墨水屏设备
- 优化墨水屏显示效果（适配低刷新率 & 灰度）
- 提供简洁、直观的信息展示界面

适用于：

- 土豆片（尚未发布，详情可关注微博@Sunbelife）
- 桌面信息卡片（天气 / 日程 / 提醒）
- 个性化信息展示终端

---

## 🚀 功能特性

- 📡 **设备连接**
  - 自动发现设备
  - 快速连接与传输

- 🖼️ **图片处理**
  - 适配墨水屏分辨率（如 400×600）
  - 支持多种图片输入
  - 针对墨水屏优化显示效果

- 🎨 **卡片展示**
  - 支持信息卡片 UI
  - 可扩展天气 / 日历 / 提醒等内容

- ⚡ **传输优化**
  - 实时进度显示
  - 后台传输支持（持续优化中）

## 🛠️ 运行方式

### 1️⃣ 克隆项目

```bash
git clone https://github.com/jiqimaooo/PotatoCardApp.git
```

---

### 2️⃣ 打开项目

使用 Xcode 打开：

```
PotatoCardApp.xcodeproj
```

---

### 3️⃣ 配置签名

在 Xcode 中：

```
Signing & Capabilities → Team
```

选择你自己的 Apple 开发者账号

---

### 4️⃣ 运行

选择真机设备，点击运行即可

---
